How To Add a new user

To add a new user click on the add user button, it is located at the top right corner just above the users table.  A sample image is shown below;

The table has different columns, each column displaying specific information about the user. The columns in the table are; Fullname,  Username,  Email,  Roles,  Status,  and Actions. 

  • Fullname: This column displays the first and last name of the user.
  • Username: This column displays the first and last name of the user.
  • Email: This column displays the first and last name of the user.
  • Roles

    The roles column indicates the current role assigned to the user, the role limits the activities of a user. The roles can be either of the following; Admin, Authors, Moderators and Examiners

  • Status

    The status column indicates whether the user is either active or inactive.
    Note:  If the status is inactive then the user will not be able to login to the application.
